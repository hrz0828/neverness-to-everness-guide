队伍模板的价值在于给你一个可执行的起点，而不是要求你完全照抄。不同账号拥有的角色、弧盘、觉醒和材料都不同，所以更重要的是理解每个位置为什么存在，然后用自己已有角色替代。

## 开荒低成本队

模板结构：

1. 稳定主C。
2. 生存或承伤位。
3. 功能副C。
4. 任意补位角色。

开荒队的核心目标是稳定，而不是极限伤害。主C负责清日常和主线，生存位保证容错，功能副C补聚怪、破韧或短爆发。薄荷这类低成本输出可以承担前期过渡，等抽到更适合的主C后再逐步转为副队或备用角色。

循环建议：先放功能技能或聚怪，再切主C输出；遇到精英怪时保留生存技能，不要把所有爆发都交在小怪身上。

## 灵属性聚怪速切队

模板结构：

1. 灵属性主C或站场输出。
2. 九原一类聚怪/速切副C。
3. 生存位。
4. 增伤或补属性角色。

这类队伍适合小怪密集、需要快速收束战场的内容。九原的价值不只是伤害，还在于把分散敌人集中到主C技能范围内。只要主C能稳定吃到聚怪窗口，整体效率通常会比单纯堆第二个输出更高。

替代思路：如果没有对应角色，就找能提供范围控制、短时间爆发或敌人位移控制的功能位。

## 暗属性 DOT 队

模板结构：

1. 安魂曲一类 DOT 主C。
2. 能延长、触发或放大持续伤害的副C。
3. 生存位。
4. 功能辅助或补伤害位。

DOT 队通常不是一轮技能立刻结束战斗，而是靠持续伤害、形态切换和稳定站场压低敌人。它适合血量较高、战斗时间较长的敌人，但在短时间清杂时可能不如爆发队直接。

循环建议：先铺设标记或持续伤害，再让主C进入稳定输出阶段；生存位要留给敌人高压阶段，避免 DOT 还没结算完就被打断。

## 光属性爆发队

模板结构：

1. 光属性主C或爆发核心。
2. 浔一类速切爆发/功能辅助。
3. 增伤或能量位。
4. 生存位。

爆发队的重点是让关键技能集中落在增益窗口内。浔这类角色的价值在于技能复现、速切爆发和功能补位；如果队伍能把主C和副C的爆发窗口对齐，就能在短时间内处理精英怪或 BOSS 破绽。

循环建议：准备阶段先放辅助技能，确认敌人被控制或进入破绽后，再交主C和副C爆发。不要在敌人即将位移或无敌时交核心技能。

## 高风险爆发队

模板结构：

1. 白藏或哈索尔一类高爆发角色。
2. 能提供保护、控制或增伤的辅助。
3. 生存位。
4. 副C补真空期。

高风险爆发队适合熟悉敌人机制后使用。它的上限高，但更怕技能被打断、走位失误和生存链断档。新手使用时，不要为了追极限伤害牺牲生存位。

循环建议：先观察敌人招式，等攻击后摇、破韧或硬直窗口再爆发。没有稳定窗口时，宁可少打一轮，也不要硬吃机制换伤害。

## 生存稳健队

模板结构：

1. 中等输出但循环稳定的主C。
2. 法帝娅一类承伤或保护角色。
3. 治疗、护盾或减伤位。
4. 功能副C。

这类队伍适合新手打高压异象、陌生 BOSS 或限时失败成本较高的内容。它可能不是最快，但能减少重复挑战。只要能稳定打完，收益往往高于频繁翻车的极限输出队。

## 替代角色规则

替代角色时按职责替代，不按稀有度替代：

- 主C缺位：找能稳定站场或爆发的输出。
- 副C缺位：找短时间能补伤害、破韧或属性覆盖的角色。
- 生存位缺位：找治疗、护盾、承伤或减伤机制。
- 辅助缺位：找聚怪、能量、增伤、控制或特殊机制。

如果替代后队伍缺了关键职责，就需要调整循环，而不是只看面板。

## 模板使用检查表

每次组队前确认：

- 主C是否有完整输出窗口。
- 辅助技能是否能覆盖主C爆发。
- 生存技能是否留给高压阶段。
- 队伍是否能处理群怪和精英怪。
- 是否存在两个角色抢同一段站场时间。
- 当前材料是否支撑这套队伍成型。

模板不是答案，职责才是答案。只要职责完整，即使角色不是最优，也能打出稳定体验。
